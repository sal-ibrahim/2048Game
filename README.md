# 2048Game


# About
2048 is a console-based recreation of the popular puzzle game, implemented in **C**. Players combine numbered tiles on a 4×4 grid to reach 2048. The project demonstrates algorithmic design and software principles while managing real-time game mechanics, tile merging, and score tracking.


## Features
- Classic 4×4 2048 gameplay
- Real-time user input handling
- Automatic tile merging and score calculation
- Detects game over and winning conditions
- Custom interface

---

## Technologies Used
- **Programming Language:** C  
- **Library:** Allegro
- **Concepts:** Algorithmic design, game mechanics, real-time interaction
